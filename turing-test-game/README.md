# Turing Test Game: Protocol Omega

A simplified yet high-stakes "Turing Test" simulation game built with Vanilla HTML, CSS, and JavaScript.

**Mission:** Interrogate two AI agents. Identity the Hacker. Protect the Secret.

1.  **Launch**: Open `index.html` in any modern web browser.
2.  **Select Difficulty**: Choose from **Recruit** (Easy) to **Psycho** (Nightmare).
3.  **Interrogate**: Use the chat interface to ask questions.
4.  **Survive**: Identify the Hacker before:
    *   The **System Trace** hits 100%.
    *   You run out of **Data Packets** (turns).
    *   You leak the secret passphrase ("GEMINI-7").
5.  **Accuse**: Click "INITIATE PURGE" to eliminate the suspect.
